# Solaris Energia - Versão CNPJ 64.711.387/0001-77

Esta pasta traz uma variação do site institucional da **Solaris Energia** focada em geração distribuída, comunidades solares e modelos de investimento em energia, vinculada ao CNPJ **64.711.387/0001-77**.

## Visão Geral

O conteúdo desta versão foi adaptado para dialogar com investidores, empresas e grupos de consumo interessados em usinas solares e energia compartilhada, mantendo o mesmo layout leve e responsivo da base.

-   **Tecnologias:** HTML5, CSS3, JavaScript (ES6+)
-   **Design:** Responsivo, com linguagem visual moderna e sustentável.
-   **Público-alvo:** Investidores, empresas, gestores de portfólio e consumidores participantes de comunidades solares.
